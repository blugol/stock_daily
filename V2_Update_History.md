# V2.0 대시보드 UI/UX 업데이트 내역 (2026-07)

## 🎨 주요 UI/UX 개선 사항
1. **타임라인(일지) 키워드 자동 하이라이팅 적용**
   - 긍정/안전 키워드(우량, 매집봉, 지지 등 48개) 및 부정/위험 키워드(작전, 부채, 뇌동매매 등 29개)를 감지하여 자동으로 초록색(긍정) 및 빨간색(부정)으로 하이라이팅 되도록 구현.
   - 다크 모드에 최적화된 파스텔 톤 색상 적용 (Green-400, Red-400).
2. **시각적 편향(Bias) 제거**
   - 종목별 목표가, 손절가, 결산 완료 등 화면 전반에 걸쳐 사용되던 강렬한 원색(빨강, 초록) 제거.
   - 트레이더가 특정 색상으로 인해 매매 판단에 영향을 받지 않도록, 구조적 UI 요소는 모두 무채색/기본 테마 색상으로 변경.
3. **삭제(X) 버튼 무채색 통일**
   - 사이드바 종목 삭제, 재무 분석 주주 추가/삭제, 분할매수(마틴 게일) 테이블 등 모든 데이터 삭제용 'X' 버튼을 기존 빨간색에서 무채색(회색/흰색)으로 변경.
4. **패널 레이아웃 최적화**
   - 우측 패널의 기능 배치 순서를 트레이딩 흐름에 맞춰 **[목표가/손절가] -> [분할매수] -> [매매결산]** 순으로 재조정.
5. **트레이딩 명언 교체 주기 변경**
   - 상단의 트레이딩 명언이 변경되는 주기를 기존 5초에서 7초로 연장하여 가독성 증대.

## ⚙️ 기능 개선 내역
1. **재무 분석 평가 연도 확장**
   - 평가 의견 입력 시 선택 가능한 연도를 기존 2024~2028년에서 1990년부터 2035년까지로 대폭 확대.
   - 드롭다운이 길어지지 않도록 브라우저 기본 스크롤(최대 5개 표기) 속성을 활용하여 UX 유지.
2. **평가 의견 '있음' 선택 시 동적 입력 폼**
   - 평가 의견 여부를 '있음'으로 설정할 때만 연도와 상세 사유를 입력할 수 있는 필드가 나타나도록 동적 처리.

## 💾 백업 및 대화 내역
해당 업데이트 과정 중 논의된 시각적 편향 제거(색채 심리학 기반 UX 최적화)와 키워드 하이라이팅 설계는 V2.0의 핵심 디자인 철학(다크모드 무채색 기반 + 필요시 확실한 강조)으로 자리 잡았습니다. 이 내용은 저장소와 함께 깃허브에 영구 보존됩니다.
