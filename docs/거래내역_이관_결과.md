## 1. 이관 작업 통계
* **대상 소스 파일**: [거래내역(과거데이터).csv](file:///c:/Users/user/Desktop/복기/거래내역(과거데이터).csv) (CP949)
* **이관 결과**: **총 10개 종목**의 매매 내역(총 19건의 상세 체결 내역)이 자동 이관 및 복기 데이터로 통합되었습니다.
* **이관된 종목 목록**: `신신제약`, `큐알티`, `링크제니시스`, `지오엘리먼트`, `그린리소스`, `엠엑스로보틱스`, `화성밸브`, `지에스이`, `비스토스`, `유니온바이오메트릭스`

---

## 2. 세부 처리 및 이관 내용
1. **마틴 계산기 자동 대입 (매수 건)**: 
   - 각 종목의 매수 단가와 매수 수량, 매수 일자를 연동하여 마틴 계산기의 1차, 2차 등에 순서대로 기입하고 자동으로 체결 완료(`checked: true`) 처리했습니다.
   - 첫 매수 수량을 마틴의 기본 수량(`base_qty`)으로 자동 세팅했습니다.
2. **실현 손익 및 자동 결산 (매도 건)**:
   - 매도 거래가 있는 종목(`엠엑스로보틱스`, `화성밸브`, `지에스이`, `비스토스` 등)은 매수 평단가와 매도 단가를 추적하여 **최종 수익률, 보유 기간(일수), 결산 텍스트 스탬프**를 자동으로 계산하여 입력했습니다.
   - 예: **엠엑스로보틱스** -> 매수평단 `5,886원` / 매도단가 `6,040원` -> 수익률 `+2.61%` (1일 보유) 결산 완료.
3. **타임라인 내역 기록**:
   - 매수와 매도 각각의 체결 시각과 내역을 타임라인 로그에 자동으로 추가했습니다. (예: `[실제 체결] 1차 매수 51주 완료 (체결단가: 5,880원)`)
   - 결산 완료 시 결산 스탬프 로그를 자동으로 생성하여 타임라인 최상단에 기록했습니다.

## 4. 대시보드 확인 방법
1. 대시보드를 새로고침하거나 켜시면, 왼쪽의 **종목 목록 사이드바**에 `엠엑스로보틱스`, `비스토스`, `지에스이` 등 새로운 종목들이 나타납니다.
2. 각 종목 이름 옆에는 결산 완료된 **수익률(`+2.61%`, `-5.74%` 등)**이 친절히 표시되며, 해당 종목을 클릭하시면 언제든 세부 마틴 상태와 매매 기록 타임라인을 확인 및 수기 편집하실 수 있습니다!
