### 핵심 로직 구조

#### 🔍 HTS 종목 감지 스레드 (`hts_monitor_thread`)
```python
# 동작 흐름
while True:
    1. win32gui.EnumWindows()로 영웅문4 메인 창 핸들(HWND) 탐색
    2. ALLOWED_WINDOW_KEYWORDS(차트, 일지, 주가...) 포함 자식창 필터
    3. EXCLUDE_WINDOW_KEYWORDS(계좌, 잔고...) 제외 처리
    4. AfxWnd110 클래스 컨트롤에서 종목명 텍스트 추출
    5. POTENTIAL_NAME_PATTERN 정규식으로 유효한 종목명 검증
    6. EXCLUDED_WORDS 무효어 필터링
    7. active_stock 전역변수 갱신 (threading.Lock 보호)
    time.sleep(1.0)
```

#### 🌐 HTTP API 엔드포인트
| 메서드 | URL | 기능 |
|---|---|---|
| `GET` | `/api/status` | 현재 감지된 종목명 반환 |
| `GET` | `/api/list` | 전체 종목 목록(파일명 기반) 반환 |
| `GET` | `/api/memo?stock=종목명` | 해당 종목 JSON 데이터 반환 |
| `POST` | `/api/memo?stock=종목명` | 해당 종목 JSON 데이터 저장 (백업 후 덮어쓰기) |
| `DELETE` | `/api/memo?stock=종목명` | 종목 파일 삭제 (휴지통→backup/ 이동) |
| `GET` | `/*` | 정적 파일 서빙 (html/css/js) |

#### 🛡️ 보안 및 안전 장치
```python
# 파일명 Sanitize (경로 탈출 공격 방지)
safe_name = "".join(x for x in stock_name if x.isalnum() or x in " -_").strip()

# 저장 전 자동 백업
def save_backup(stock_name, filepath):
    timestamp = time.strftime("%Y%m%d_%H%M%S")
    shutil.copy2(filepath, f"backup/{timestamp}_{safe_name}.json")
```

```typescript
// settlement 필드의 타입 정의 (TypeScript 스타일)
interface Settlement {
    final_price: number;       // 최종 매도 단가
    is_settled: boolean;       // 결산 완료 여부
    holding_period: string;    // "10D" 형식
    yield_rate: string;        // "+23.88%" 형식
    stamp: string;             // legacy 텍스트 스탬프 (호환용)
    sell_date: string;         // "YYYY-MM-DD" 형식
}

interface Tier {
    tier: number;      // 차수 (1, 2, 3...)
    ratio: number;     // 마틴 배수 (1, 1, 2, 4...)
    price: number;     // 매수 단가
    qty: number;       // 실제 매수 수량
    checked: boolean;  // 매수 집행 여부
    date: string;      // 매수 집행일 "YYYY-MM-DD"
}
```

## 7. 보안 제약사항 및 실행 조건

> [!IMPORTANT]
> `server.py`는 반드시 **트레이더님이 직접 본인 PC에서 실행**해야 합니다.
>
> HTS 창 감지(`win32gui.GetWindowText`)는 **실제 로그인 데스크탑 세션**에서만 작동합니다.
> 백그라운드 서비스나 원격 세션에서 실행하면 창 목록이 비어 있어 종목 감지가 불가합니다.

### 실행 방법
```bash
# CMD 또는 PowerShell에서 직접 실행
cd C:\Users\user\Desktop\복기
python server.py

# 또는 bat 파일 더블클릭
실행하기.bat
```

### 의존성 설치 (최초 1회)
```bash
pip install pywin32
```
