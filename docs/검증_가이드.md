# 검증 가이드 (HTS 연동 복기 메모장 v2.0)

트레이더 요구사항 V2.0에 따라 **로컬 오프라인 웹 기반 복기 대시보드**의 모든 핵심 기능 개발을 완료했습니다.

---

## 1. 생성 및 적용된 파일들

* **[server.py](file:///c:/Users/user/Desktop/복기/server.py)**: 로컬 API 서버 겸 HTS 종목 모니터링 모듈 (윈도우 API로 HTS를 1초마다 감시)
* **[index.html](file:///c:/Users/user/Desktop/복기/index.html)**: 전술 대시보드 UI 구조 (영웅문 전용 다크 테마 구조)
* **[style.css](file:///c:/Users/user/Desktop/복기/style.css)**: 눈이 편안한 텍스트 가독성 최적화 다크 모드 스타일
* **[app.js](file:///c:/Users/user/Desktop/복기/app.js)**: 마틴 계산, 동적 노션 태그, 자동 결산, 타임라인 로그 제어 스크립트
* **[실행하기.bat](file:///c:/Users/user/Desktop/복기/실행하기.bat)**: 통합 실행 배치 파일

---

## 2. 사용 및 테스트 방법

1. **프로그램 구동**:
   - 작업 폴더 내의 **[실행하기.bat](file:///c:/Users/user/Desktop/복기/실행하기.bat)** 파일을 더블 클릭하여 실행합니다.
   - 검은색 터미널 창이 뜨며 서버가 구동되고, **자동으로 웹 브라우저가 열리며 복기 대시보드(`http://localhost:8000`)**가 표시됩니다.

2. **HTS 연동 테스트**:
   - 영웅문4 HTS를 켭니다.
   - HTS에서 종목을 클릭하여 변경(예: 삼성전자, 윤성에프앤씨 등)하면, 브라우저 상단 중앙의 배지가 **해당 종목명으로 실시간 자동 전환**되는지 확인합니다.

3. **기능 작동 테스트**:
   - **추세 상태**: 원하는 차트 상태(예: `눌림목 지지중`)를 클릭하면 상태에 맞게 테두리 색상이 강조됩니다.
   - **동적 태그**: 입력창에 단어(예: `자사주10%`)를 입력하고 **Enter**를 치면 알약 모양의 배지가 생깁니다. (`x` 버튼을 누르면 즉시 삭제)
   - **마틴 계산기**: `1차 진입 단가`와 `수량`을 입력하면 2차, 3차, 4차의 권장 수량과 단가가 1-1-2-4 배수로 자동 표기됩니다.
     - 각 차수별로 `[ ] 양봉 확인` 체크박스를 체크하면 오늘 날짜가 자동으로 기록되며, **현재 실시간 평단가**가 자동으로 계산되어 노출됩니다.
   - **매매 자동 결산**: 매매가 종료되면 `최종 매도 단가`를 입력하고 **[최종 청산]** 버튼을 누릅니다.
     - 1차 매수일부터 오늘까지의 **보유 기간(D)**과 평단가 대비 **수익률(%)**이 즉시 계산됩니다.
     - 하단에 생성된 **결산 스탬프** (예: `0611(5240) - 0612(6070) / (1D) / (+15.61%)`)를 클릭하면 클립보드에 자동 복사됩니다.
   - **타임라인 피드**: 하단 텍스트 창에 감정이나 차트 분석을 적고 **Enter**를 치면, 트위터처럼 날짜와 시간이 찍힌 채 타임라인에 누적 보관됩니다.

4. **저장 상태 확인**:
   - 종목을 다른 것으로 바꿨다가 다시 돌아와 보면 작성했던 내용들이 완벽하게 복구되어 있습니다.
   - 모든 데이터는 작업 폴더 내의 `data/[종목명].json` 파일로 안전하게 자동 저장되어 있습니다.
