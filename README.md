# 영웅문 연동 복기 대시보드 v2.0

키움증권 영웅문4(HTS)와 실시간 연동되는 **로컬 오프라인 트레이딩 복기 대시보드**입니다.  
HTS에서 종목을 바꾸면 대시보드가 자동으로 해당 종목의 복기 일지를 열어줍니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🔄 **실시간 HTS 연동** | 영웅문4에서 종목 변경 시 대시보드 자동 전환 |
| 📊 **마틴 분할매수 계산기** | 1-1-2-4 비율 자동 연산 + 진입 비중 분석 |
| 🏆 **매매 자동 결산** | 보유기간·수익률 자동 계산 + 4칸 그리드 표시 |
| 💬 **타임라인 메모** | 시간별 블록 누적 일지 기록 |
| 🛡️ **자동 백업** | 수정·삭제 시 backup/ 폴더에 자동 보관 |

---

## 실행 방법

### 1. 필수 패키지 설치 (최초 1회)
```bash
pip install pywin32
```

### 2. 서버 실행
```bash
# CMD에서 직접 실행
python server.py

# 또는 bat 파일 더블클릭
실행하기.bat
```

### 3. 브라우저에서 접속
```
http://localhost:8000
```

> ⚠️ **중요**: `python server.py`는 반드시 **본인 PC에서 직접** 실행해야 합니다.  
> 영웅문4 HTS 창 감지는 실제 로그인 데스크탑 세션에서만 작동합니다.

---

## 기술 스택

- **Backend**: Python 3.x (`http.server`, `win32gui`)
- **Frontend**: Vanilla HTML + CSS + JavaScript (프레임워크 없음)
- **데이터 저장**: 로컬 JSON 파일 (`data/[종목명].json`)
- **HTS 감지**: Windows API `GetWindowText` via `pywin32`

---

## 폴더 구조

```
복기/
├── server.py          # 웹서버 + HTS 감지 엔진
├── index.html         # 대시보드 UI
├── app.js             # 프론트엔드 로직
├── style.css          # 다크모드 스타일
├── 실행하기.bat       # 실행 배치 파일
├── data/              # 종목별 복기 JSON (git 제외)
├── backup/            # 자동 백업 (git 제외)
├── docs/              # 개발 문서
└── tools/             # 유틸리티 스크립트
```

---

## 주의사항

- `data/`, `backup/`, `raw_data/` 폴더는 개인 트레이딩 기록이므로 `.gitignore`로 제외되어 있습니다.
- 처음 실행 시 `data/`, `backup/` 폴더가 자동 생성됩니다.
